# CUTIE GETS A GIFT

A simple gift-themed web page project built with HTML, CSS, and JavaScript.

## Pages

- `index.html` — Landing page with a secret code modal to reveal the gift.
- `welcome.html` — Celebration page shown after the correct code is entered.

## How to run

1. Open `index.html` in your browser.
2. Click **Open Gift**.
3. Enter the secret name `sulu` and code `1234`.
4. The page redirects to `welcome.html`.

## Files

- `index.html` — main landing page
- `welcome.html` — gift celebration page
- `styless.css` — shared styles for both pages
- `welcome.css` — styles specific to the welcome page
- `script.js` — sign-in modal behavior
- `welcome.js` — page load behavior for the welcome page

## Notes

This project is designed as a cute interactive landing page and can be viewed locally in any browser.
